**KCD2 ModForge** is a Pure Java application designed to help modders of *Kingdom Come: Deliverance 2* easily read,
edit, and export XML-based game files into fully playable mods.
**KCDTextureExporter**
A tool for converting KCD1's and KCD2's texture files into TIF or raw DDS files.

---

## Features

- Import, edit, and create Game Data, Localizations, and more
- Mod Management: use files for import-export
- Automatic generation of mod folder structure including manifest compatible with other KCD2 modding tools
- GameEngine config management ... idk why would you use that over the autoexec.cfg tho ...
- Export mods as Folder containing `.pak` files ready to be used in the game
- Import mods from folders
- Seamless workflow from creating a new mod to editing elements
